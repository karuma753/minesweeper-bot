# マインスイーパーボット

以前作成したマインスイーパーを自動解析プログラムを発展させ、デスクトップ上で認識・操作させるプログラムです。

## このプロジェクトのポイント
- ターミナル上ではなく、実際のPCの画面を画像認識（テンプレートマッチング）を用いて画面上の盤面を分析し、マウスの操作をもってクリアを目指す

## 使用技術
- Python 3
- PyAutoGUI（画像認識・操作）
- OpenCV（画像処理）
